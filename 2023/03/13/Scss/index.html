3.5混合模式

语句嵌套

语句带参数

  1. 混合指令 ( @mixin ) 是可以重复使用的一组 Css 声明；
  2. 有助于减少重复代码，只需声明一次就可以在文件中反复引用；
  3. 包含所有 Css 规则以及绝大部分 Sass 规则，甚至通过参数引入变量；
  4. 使用参数时，建议加上默认值。
